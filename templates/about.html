Purpose

The portal is designed to help students find study materials, notes, and reference documents in one place. Teachers (admins) can upload verified resources to ensure high-quality content.

Features

  • Browse and search study resources by category.
  • Upload resources (superusers only).
  • User-friendly interface with secure login and registration.

Tech Stack

This project is built using Django (Python) for the backend, Bootstrap 5 for styling, and SQLite as the database. It demonstrates basic CRUD operations, authentication, and file handling.

This portal is a mini-project for academic purposes and can be extended to a full-fledged student resource management system in the future.
